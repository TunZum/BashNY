# ПиТПМ2 - Башня

Башня состоит из 8 этажей.
Боссы на 4 и 8 этажах выбираются рандомно из 2, которые находятся в соответствующей категории.
Прописать 5 категорий вражеский юнитов по 10 классов (3 основных (1, 2 и 3 уровень этажа), 3 уникальных (5, 6, 7 уровень этажа) и 4 босса (2 между 4 этажа и 2 после 8 этажа)).
Прописать 5 категорий предметов брони (Шлем, Грудь, Перчатки, Штаны, Ботинки) по 15 предметов с 3 видами редкости (Простой, Уникальный, Редкий)
Прописать 5 категорий предметов оружия (Меч, Кинжал, Дубина, Эсток, Топор) по 15 предметов с 3 видами редкости (Простой, Уникальный, Редкий)
Основные характеристики:
  o	Урон – наносимый врагам урон по здоровью.
  o	Броня – дополнительное здоровье, поверх основного.
  o	Здоровье – количество очков до смерти.
  o	Скорость атаки – время между атаками.
  o	Крит – множитель урона.
  o	Шанс крита – шанс вызова крита.
  o	Уклонение – вероятность избежание урона полностью.
Базовое значение:
  o	Урона = 5.
  o	Здоровья = 25.
  o	Броня = 0.
  o	Скорость атаки = 1 сек/1000мс.
  o	Множитель крита = 1.1х.
  o	Шанс крита = 5%.
  o	Уклонение = 5%.
При нанесении урона свыше количество количества брони, остаток отнимается от здоровья.
Для увеличения скорость атаки снижается её значение
При снаряжении оружие, его значения переписывают базовое значение персонажа (Атака, скорость атаки, шанс крита, крит).
При снаряжении брони, их значения дополняют базовые значения персонажа.
Броня и Здоровье восполняются полностью между противниками.
При победе над противником дается выбор из 3 случайных снаряжений (может выпасть любой категории) соответствующему уровню этажа.
Все элементы распологаются в panel1, в котором находится фоном изображение локации (4 локации: 1-3 этаж, босс1, 5-7 этаж и босс2).
2 picterBox по бокам (отображение героя и врага) и в центре richTextBox (основная информация).
Скрытые 3 кнопки для выбора лута.
2 timer (скорость атаки героя и скорость атаки врага)
